# Support And Services

Enterprise Hardening Toolkit is open source.

Community users can use GitHub issues for bugs, documentation gaps, and safe feature requests.

Organizations that need help can build services around the project, including:

- authorized endpoint hardening audits
- baseline customization
- remediation planning
- internal security reporting
- compliance preparation
- dashboard and PDF export extensions
- enterprise deployment design

Do not share secrets, credentials, private customer data, or unauthorized system data in public support requests.
